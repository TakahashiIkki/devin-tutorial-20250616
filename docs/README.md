# PINコード認証システム with Mailpit

## プロジェクト概要

Mailpit Docker コンテナを使用してメール送受信機能を持つPINコード認証システムを構築する。
JavaScriptベースのWebアプリケーションで、メールアドレスによる認証とPINコード検証を実装する。

## 機能要件

### 1. ログイン（メールアドレス入力）ページ
- メールアドレス入力フォーム
- PINコード送信ボタン
- フォームサブミット時にメールアドレスにPINコードを送信
- PINコード入力ページへ遷移

### 2. ログイン（PINコード入力）ページ  
- PINコード入力フォーム
- ログインボタン
- PINコード検証機能
- 認証成功時はログイン済みページへ遷移

### 3. ログイン済みページ
- セッション管理（Cookie）
- ログイン状態の判定
- 未ログイン時はアクセス制限

## 技術要件

### 開発環境
- **フレームワーク**: Next.js 15+ (App Router)
- **言語**: TypeScript
- **メールサーバー**: Mailpit (Docker)
- **認証**: サーバーサイドアクション + セッション管理
- **スタイリング**: PoCなので一旦今はいらない!!
- **メール送信**: Nodemailer

### システム構成
```
├── app/                    # Next.js App Router
│   ├── login/             # ログインページ
│   ├── pin/               # PINコード入力ページ  
│   ├── dashboard/         # ログイン済みページ
│   └── actions/           # サーバーサイドアクション
├── lib/                   # ユーティリティ
├── docker/                # Mailpit設定
└── docs/                  # ドキュメント
```

## Mailpit設定
- Docker Composeでコンテナ起動
- SMTP: localhost:1025
- Web UI: localhost:8025
- メール送受信テスト機能

## 実装詳細

### サーバーサイドアクション
- `sendPinCode()`: メールアドレスにPINコード送信
- `verifyPinCode()`: PINコード検証とセッション作成
- `checkSession()`: ログイン状態確認

### ページ構成
- `/login`: メールアドレス入力フォーム
- `/pin`: PINコード入力フォーム
- `/dashboard`: ログイン済みコンテンツ
- `/`: リダイレクト処理
